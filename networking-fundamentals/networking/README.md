# Networking Concepts

This section dives into essential networking topics for cybersecurity professionals. Understanding networking protocols and configurations is key to becoming proficient in penetration testing.

---

## Topics covered

* **IP Addressing**: Understanding IPv4, CIDR notation, and subnetting.
* **Subnetting**: Breaking down networks into smaller, manageable subnets.
* **Routing**: Understanding how data travels across networks.
* **ARP (Address Resolution Protocol)**: Resolving IP addresses to MAC addresses.
* **ICMP (Internet Control Message Protocol)**: Basics of ICMP and how it's used in network operations (including ping).
* **DNS (Domain Name System)**: Translating domain names into IP addresses.

---

## Why these topics matter

* **IP Addressing & Subnetting**: Crucial for network discovery and enumeration during penetration testing.
* **Routing & ARP**: Understanding how systems communicate with each other on a network.
* **ICMP & DNS**: Important for reconnaissance and enumeration tasks.
