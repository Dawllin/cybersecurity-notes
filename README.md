# Networking Fundamentals

This section covers the core networking concepts required for cybersecurity and penetration testing.

Understanding these concepts is essential for network scanning, enumeration, and exploitation.

---

## Topics covered

* Bits and binary representation
* Binary to decimal conversion
* Subnet mask
* CIDR notation
* Host calculation
* Block size
* Subnetting
* Network flow

---

## Skills developed

* Understanding IP addressing
* Calculating hosts per subnet
* Interpreting CIDR notation
* Identifying network ranges
* Understanding how network communication works

---

## Why this matters in cybersecurity

Attackers use networking knowledge to:

* Identify target networks
* Calculate valid host ranges
* Perform network scanning
* Enumerate systems

Networking fundamentals are the foundation of penetration testing.
